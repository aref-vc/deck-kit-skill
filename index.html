Deck Kit

Fifteen editorial slide templates, as self-contained HTML.

One backbone, fifteen distinct looks. Each deck opens in any browser, edits in place, swaps its own images, and exports a pixel-accurate PDF. No install, no server.

Each card opens a live, empty template, or the same Claude Fable 5 report rendered in that style. Full source on GitHub.

Built with the Deck Kit. Fonts are open-licensed (SIL OFL / Apache 2.0). See the README and skill spec.
